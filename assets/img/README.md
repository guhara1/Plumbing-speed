# 시공 사진 넣는 방법 (총 21장)

이 폴더에 사진 파일을 **`01.webp` ~ `21.webp`** 이름으로 넣으면
메인 페이지 갤러리와 모든 지역 페이지에 자동으로 사진이 표시됩니다.

- 파일명: `01.webp`, `02.webp`, … `21.webp` (2자리 숫자, 확장자 `.webp`)
- 권장 비율: 가로:세로 = 16:10 (예: 1000×625px), 파일당 50KB 내외로 최적화
- 사진이 아직 없는 번호는 회색 자리(placeholder)가 대신 보이고, 파일을 넣는 즉시 사진으로 바뀝니다.
- 현재 21장은 실제 시공 사진(webp, 약 45~52KB)으로 채워져 있습니다.

각 번호의 사진 주제(캡션·alt)는 `data/images.js` 에 정의되어 있으며, 아래 표는 실제 사진 내용과 일치합니다.
주제를 바꾸고 싶으면 `data/images.js` 의 `alt`/`caption` 을 수정하세요.
