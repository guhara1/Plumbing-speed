# 시공 사진 넣는 방법 (총 21장)

이 폴더에 사진 파일을 **`01.jpg` ~ `21.jpg`** 이름으로 넣으면
메인 페이지 갤러리와 모든 지역 페이지에 자동으로 사진이 표시됩니다.

- 파일명: `01.jpg`, `02.jpg`, … `21.jpg` (2자리 숫자, 확장자 `.jpg`)
- 권장 비율: 가로:세로 = 16:10 (예: 800×500px 이상), 파일당 300KB 내외로 최적화
- 사진이 아직 없는 번호는 회색 자리(placeholder)가 대신 보이고, 파일을 넣는 즉시 사진으로 바뀝니다.

각 번호의 사진 주제(캡션·alt)는 `data/images.js` 에 정의되어 있습니다.
주제를 바꾸고 싶으면 `data/images.js` 의 `alt`/`caption` 을 수정하세요.
